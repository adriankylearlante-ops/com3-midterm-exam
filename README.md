The program is called the Dorm Room Chore Management System and is designed to manage and monitor dorm room chores. It allows the user to input the room monitor’s name and room number, then displays a list of tasks with their schedule. The user can assign up to four chores by choosing a task number, entering the roommate’s name, and specifying if the task is done or not. After all inputs are completed, the program calculates the completion rate and generates a detailed weekly chore report along with the overall room status.
